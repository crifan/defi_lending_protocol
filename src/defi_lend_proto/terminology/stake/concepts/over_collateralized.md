# 过度抵押

* DeFi领域： Over-collateralized=过度抵押 
  * 背景 
    * 抵押物往往都是加密货币
    * 加密货币价格：变动往往很大 == 价格不稳定 
      * 当抵押物价格下跌时，可能都不够贷款 
        * 使用加密资产作为抵押物就是有风险的 
  * 含义：你只能借到抵押物价值 的一部分 == 不到100% 
    * 举例 
      * Compound：75% 
        * 给抵押物的价格变动流出了15% 的缓冲区 
        * 万一下跌到一定程度，有风险时，触发：清算 
          * 详见：清算Liquidation
      * Sileo：50%
